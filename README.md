# vuejsorgua-bot
Телеграм-бот для VueJS Ukraine

1. Клонуйте цей репозиторій
2. Встановіть всі залежності:
   ```shell
   pip install -r requirements.txt
   ```
3. Запустіть бота:
   ```shell
   python3 main.py
   ```

Бот запуститься в фоновому режимі. Щоб його зупинити, запустіть команду:
```shell
kill $(cat pid.txt)
```
